## 🚀 使用方法

### 啟動應用程序
```bash
python -m streamlit run main.py --server.port 8502
```

### 瀏覽缺勤原因管理頁面
1. 在側邊欄點擊「⚠️ 缺勤原因管理」

### 新增缺勤原因
1. 點擊「➕ 新增原因」按鈕
2. 在彈窗左側填寫原因名稱
3. 點擊「💾 儲存」保存

### 搜尋缺勤原因
1. 在搜尋框輸入關鍵字
2. 結果會實時過濾

### 編輯缺勤原因
1. 在原因行的「✏️ 編輯」按鈕上點擊
2. 在彈窗中修改名稱
3. 點擊「💾 儲存」保存更改

### 刪除缺勤原因
1. 在原因行的「🗑️ 刪除」按鈕上點擊
2. 查看確認 popover 中的引用檢查結果
3. 如果沒有被引用，點擊「✅ 確認」刪除
4. 如果已被引用，系統會顯示警告並阻止刪除

## 🔍 刪除保護機制詳解

### 檢查流程
```
用戶點擊刪除按鈕
        ↓
設置 session state 標記
        ↓
顯示確認 popover
        ↓
調用 get_absent_reason_attendance_count()
        ↓
檢查返回值：
  - 如果 > 0：顯示警告「無法刪除：該原因已被 X 筆考勤記錄引用」
  - 如果 = 0：允許刪除
        ↓
用戶點擊確認或取消
```

### 錯誤處理
- 如果嘗試刪除已被引用的原因，系統會：
  1. 捕獲數據庫返回的錯誤訊息
  2. 使用正則表達式提取引用次數
  3. 顯示友好的繁體中文錯誤提示
  4. 保持 popover 打開狀態讓用戶取消操作
