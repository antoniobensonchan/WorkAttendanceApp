# 🚀 快速开始指南

## ⚡ 3步启动系统

### 第1步：安装依赖（仅首次需要）
```bash
pip install -r requirements.txt
```

### 第2步：启动服务器
**方法A - 推荐：** 双击 `start.bat`  
**方法B - 命令行：** 
```bash
streamlit run main.py --server.port=8501 --server.address=0.0.0.0
```

### 第3步：访问系统
- **本机访问**：http://localhost:8501
- **其他电脑**：http://[服务器IP]:8501

---

## 📖 5分钟上手教程

### 1️⃣ 添加员工（30秒）
1. 点击侧边栏「员工管理」
2. 展开「➕ 新增员工」
3. 输入姓名和公司
4. 点击「保存」✅

### 2️⃣ 添加工地（20秒）
1. 点击侧边栏「工地管理」
2. 展开「➕ 添加工地」
3. 输入工地名称
4. 点击「保存」✅

### 3️⃣ 录入考勤（每人10秒）
1. 点击侧边栏「考勤记录」
2. 选择员工和工地
3. 勾选出勤时段（默认已勾选）
4. 输入加班小时（如有）
5. 点击「提交」✅

### 4️⃣ 查看统计（实时）
- 点击「员工统计看板」→ 查看个人统计
- 点击「工地统计看板」→ 查看工地统计

## 💡 常见问题速查

### ❓ 其他电脑无法访问？
**解决：**
1. 确认在同一局域网
2. 查看服务器IP：运行 `ipconfig`
3. 检查防火墙设置
4. 地址格式：`http://192.168.x.x:8501`

### ❓ 如何修改端口？
```bash
streamlit run main.py --server.port=8080 --server.address=0.0.0.0
```

### ❓ 数据保存在哪？
项目根目录的 `attendance.db` 文件

### ❓ 如何备份数据？
复制 `attendance.db` 文件到其他位置

### ❓ 可以多人同时操作吗？
✅ 可以！支持多用户并发

### ❓ 需要联网吗？
❌ 不需要，纯本地运行
