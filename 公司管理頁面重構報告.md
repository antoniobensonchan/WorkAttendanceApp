## ✅ 重構完成項目

### 1. 頂部工具列 (Toolbar)
已使用 `st.columns([3, 1, 1])` 佈局重新設計：

- **第一欄（寬度3）**：搜尋框，用於過濾公司名稱
- **第二欄（寬度1）**：「新增公司」按鈕，點擊後觸發寬版彈窗
- **第三欄（寬度1）**：「導入」popover，內含 Excel 模板下載與文件上傳組件

### 2. 寬版彈窗設計 (Large Dialog)
所有對話框均設置 `width="large"` 參數：

#### 新增/編輯公司彈窗
- 內部使用 `st.columns([1, 1])` 雙欄佈局
- 左側：公司名稱輸入框
- 右側：預留擴展字段空間（顯示「更多字段待擴展」）
- 底部：右對齊的操作按鈕（取消/儲存）

#### 導入彈窗 (Popover)
- 內部使用 `st.columns([2, 3])` 佈局
- 左側：操作指南與模板下載（青色漸變背景）
- 右側：文件上傳器（黃色漸變背景）
- 清晰的視覺分隔與說明文字

### 3. 列表展示邏輯
- **Pandas 模糊過濾**：根據搜尋框內容進行大小寫不敏感的模糊匹配
- **數量限制**：僅渲染過濾後的前 50 筆記錄
- **超量提示**：若超過 50 筆，在底部顯示黃色警告框：「⚠️ 僅顯示前 50 筆記錄，請使用搜尋縮小範圍（共 X 筆）」
- **默认排序**：默認按 ID 降序排列，最新添加的公司顯示在最前面

### 4. 行內操作
每一行使用 `st.columns([1, 4, 1, 1])` 佈局：
- **第 1 欄**：公司 ID（例如 #1, #2）
- **第 2 欄**：公司名稱（突出顯示，字體較大）
- **第 3 欄**：「✏️ 編輯」按鈕，觸發寬版編輯彈窗
- **第 4 欄**：「🗑️ 刪除」按鈕，觸發二次確認 popover

### 5. 刪除確認機制
- 點擊刪除按鈕後設置 session state 標記
- 顯示確認 popover，包含公司名稱和關聯員工數量警告
- 如果公司有員工關聯，顯示警告並阻止刪除
- 提供「✅ 確認」和「❌ 取消」兩個按鈕

### 6. Apple 極簡風格
- 減少背景色塊的使用
- 增加留白和間距
- 清晰的字體層次結構
- 使用細微邊框代替厚重背景
- 僅在必要處使用漸變色（導入區域）

## 🚀 使用方法

### 啟動應用程序
```bash
python -m streamlit run main.py --server.port 8502
```

### 瀏覽公司管理頁面
1. 在側邊欄點擊「🏢 公司管理」

### 新增公司
1. 點擊「➕ 新增公司」按鈕
2. 在彈窗左側填寫公司名稱
3. 點擊「💾 儲存」保存

### 搜尋公司
1. 在搜尋框輸入關鍵字
2. 結果會實時過濾

### 編輯公司
1. 在公司行的「✏️ 編輯」按鈕上點擊
2. 在彈窗中修改名稱
3. 點擊「💾 儲存」保存更改

### 刪除公司
1. 在公司行的「🗑️ 刪除」按鈕上點擊
2. 在確認 popover 中確認（如果沒有關聯員工）

### 導入公司
1. 點擊「📥 導入」popover
2. 左側：下載 Excel 模板
3. 填寫模板後，在右側上傳文件
4. 點擊「✅ 開始導入」
