   - 第一次執行時，系統會在您的「文件」資料夾自動建立資料庫

   - 您不需要手動做任何事情！

#### 如果遇到問題

**問題 1：仍然出現「唯讀」錯誤**

解決方法：
```
1. 檢查您的用戶名稱是否有特殊字元或空格
2. 確認「文件」資料夾存在於：C:\Users\[您的名稱]\Documents
3. 以系統管理員身分執行一次應用程式
```

**問題 2：找不到資料庫**

解決方法：
```
1. 打開檔案總管
2. 前往：C:\Users\[您的名稱]\Documents\WorkAttendanceApp
3. 確認 attendance.db 檔案存在
4. 如果不存在，刪除整個 WorkAttendanceApp 資料夾重新執行 .exe
```

**問題 3：想要備份資料**

解決方法：
```
1. 關閉應用程式
2. 複製以下檔案到其他位置（例如 USB 隨身碟）：
   C:\Users\[您的名稱]\Documents\WorkAttendanceApp\attendance.db
3. 需要還原時，將檔案複製回相同位置
```
