# 快速使用指南

## 🚀 三步启动系统

### 第一步：安装依赖（首次使用）
```bash
pip install -r requirements.txt
```

### 第二步：启动服务器
**方法1 - 使用启动脚本（推荐）：**
```
双击 start.bat
```

**方法2 - 命令行启动：**
```bash
streamlit run main.py --server.port=8501 --server.address=0.0.0.0
```

### 第三步：访问系统
- **本机访问**：http://localhost:8501
- **其他电脑访问**：http://[服务器IP]:8501

## 📖 操作流程

### 初次使用设置流程

1. **添加员工**
   - 点击侧边栏「员工管理」
   - 展开「➕ 新增员工」
   - 输入员工姓名和所属公司
   - 点击「保存」

2. **添加工地**
   - 点击侧边栏「工地管理」
   - 展开「➕ 添加工地」
   - 输入工地名称
   - 点击「保存」

3. **录入考勤**
   - 点击侧边栏「考勤记录」
   - 展开「📝 新增考勤记录」
   - 选择日期、员工、工地
   - 勾选上午/下午/晚上出勤
   - 输入加班小时数（如有）
   - 点击「提交」

4. **查看统计**
   - 员工统计看板：查看每个员工的总出勤情况
   - 工地统计看板：查看每个工地的每日出勤情况

## 💡 常见问题

### Q1: 其他电脑无法访问系统？
**解决方法：**
1. 确保所有电脑在同一局域网
2. 检查防火墙设置，允许端口8501
3. 确认服务器IP地址正确
4. 尝试关闭Windows防火墙测试

### Q2: 如何查看服务器IP地址？
**Windows系统：**
```bash
ipconfig
```
找到「IPv4 地址」即为服务器IP

### Q3: 数据保存在哪里？
数据保存在项目根目录的 `attendance.db` 文件中

### Q4: 如何备份数据？
复制 `attendance.db` 文件到其他位置即可

### Q5: 可以多人同时操作吗？
可以，系统支持多用户同时访问和操作

### Q6: 需要联网吗？
不需要，系统完全本地运行，只需局域网连接

## 🎯 使用技巧

### 管理员日常操作
1. **每天早上**：录入当天考勤
2. **每周查看**：员工统计看板，了解出勤情况
3. **每月导出**：通过工地统计看板汇总数据

### 快速录入考勤
- 日期默认为今天，无需修改
- 出勤默认全部勾选，只需取消缺勤时段
- 加班小时默认为0，有需要再修改

### 数据查询
- 员工统计：自动计算，实时查看
- 工地统计：可按工地和日期范围筛选

## ⚙️ 高级设置

### 修改端口号
如果8501端口被占用，可修改为其他端口：
```bash
streamlit run main.py --server.port=8080 --server.address=0.0.0.0
```

### 设置固定IP
建议为服务器设置固定IP，方便其他电脑访问

### 开机自启动
将 `start.bat` 的快捷方式放到启动文件夹
